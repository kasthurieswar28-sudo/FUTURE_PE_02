# AI Portfolio Website

## Project info

**URL**: https://kasthurieswarportfolio.lovable.app

This project is a professional AI portfolio website created using **Lovable**.
It is designed to showcase skills, projects, and contact information for students, interns, or freelancers.

## Project Overview
The portfolio highlights my interest in Artificial Intelligence, Prompt Engineering, and modern web tools.
It provides a clean and simple interface to present my profile and work.

## Features
- Single-page portfolio layout
- Professional hero and about sections
- Skills and projects showcase
- Contact form for easy communication
- Built using AI-assisted website generation

## Tools Used
- Lovable (AI website builder)
- AI-generated content and layout assistance

## Purpose
This project was developed as part of an internship task to demonstrate the ability to use AI tools for building a functional and professional portfolio website.
